Описание работы скриптов для Диалога НПС.


DialogueManager Необходимо вешать на пустой объект на сцене. 
Он управляет системой диалога в целом. 
Метод StartDialogue() вызывается на UI-кнопке "Начать говорить".
Метод DisplayNextSentence() вызывается на UI-кнопке "Далее".

DialogueTrigger Необходимо вешать на триггер с нпс, который будет говорить. 
Он вызывает UI-кнопку для начала диалога и подает все данные в Диалог Менеджер (DialogueManager)
